● 项目技术介绍

  1. 核心语言

  - Python 3.x - 主要编程语言

  2. 数据获取

  - requests - HTTP请求库，用于调用B站搜索API
  - session管理 - 维持cookie和请求头
  - User-Agent伪造 - 模拟浏览器访问
